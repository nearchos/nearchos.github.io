Software

On GitHub

I love developing software, especially mobile apps with a backend side. My work includes some purely research oriented projects as well as some more down to earth software.

active

  • I have developed Cyprus Pharmacy Guide as a toy project when I was learning Android. Since then, I have published it on Google Play Store and it has become quite popular receiving praising comments from its users. I have not managed to make its source code available yet, but I intend to do so.
  • The Code Cyprus app is an Android app, which realizes the client-side of a gamified treasure hunt. This app is used as the official client of the yearly Code Cyprus Treasure Hunt (which usually involves ~100 participants). Its code is hosted on GitHub as open source (under LGPL license).
  • The Treasure Hunt Challenge is a server-side software, implemented on Google App Engine, used to run gamified treasure hunts. This system is used as part of the Computing Skills module at UCLan Cyprus, and also as the server side of the yearly Code Cyprus Treasure Hunt (which usually involves ~100 participants). Its code is hosted on GitHub as open source (under LGPL license).
less active
  • I have developed the Really Simple Context Middleware (RSCM) project, originally hosted on Google Code, and now on GitHub. It is intended to be a framework (pluggable, middleware architecture) facilitating the development of context-aware apps for Android.
